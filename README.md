# クリエイターポートフォリオ

静的なシングルページのポートフォリオサイトです。`index.html` をブラウザで開くだけで閲覧できます。

## 構成

- `index.html` メインのページ
- `css/style.css` スタイル（ライト/ダークテーマ対応）
- `js/main.js` インタラクション（モバイルメニュー、スムーススクロール、テーマ切替、トップへ戻る）
- `assets/` 画像等（必要に応じて追加）

## 使い方

1. `index.html` をブラウザで開く
2. `Projects` セクションに実績を追加し、リンクや説明文を編集
3. `Your Name` をご自身の名前に変更
4. 必要に応じて色やフォントを `css/style.css` の変数で調整

## デプロイ

- GitHub Pages で `index.html` を公開するか、Netlify/Vercel などにドラッグ&ドロップでデプロイできます。
- お問い合わせフォームは Netlify Forms を想定した簡易設定（`netlify` 属性と `form-name`）になっています。Netlify を使用しない場合は外部フォームサービスやメールリンクに置き換えてください。
