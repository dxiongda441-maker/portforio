# Alter Portfolio

オルタのポートフォリオサイトです。

自然、四季、日本的な雰囲気を背景に、AI音楽、AIアニメ、AIゲーム、アプリ開発の制作実績を見せるシングルページ構成です。

## 公開URL

- GitHub Pages: https://dxiongda441-maker.github.io/portforio/
- GitHub repository: https://github.com/dxiongda441-maker/portforio

## 編集方法

- 公開ページは `main` ブランチの `index.html` です。
- 修正は作業ブランチで行い、`main` に反映するとGitHub Pagesが自動更新されます。
- Next.js版のソースはローカルの `app/`、`package.json`、`next.config.mjs` にあります。
