# Alter Portfolio

SpaceXのような大きなビジュアル主導の構成を、日本の自然、四季、伝統文化の空気感に置き換えたポートフォリオです。

## 公開URL

- GitHub Pages: https://dxiongda441-maker.github.io/portforio/
- GitHub repository: https://github.com/dxiongda441-maker/portforio

## 使い方

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 差し替える場所

`app/page.tsx` の `profile` を編集すると、名前、肩書き、メール、SNSリンクを変更できます。作品欄は `works`、スキル図は `skills` から変更できます。

GitHub Pages版はリポジトリの `main` ブランチにある `index.html` です。修正を `main` に反映すると公開ページも自動更新されます。
